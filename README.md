# PM Portfolio Starter

A beginner-friendly Product Manager portfolio starter built with Next.js and Tailwind CSS.

## Pages included

- Home page
- About page
- Four case study pages generated from reusable project data

## Edit content

- Personal details: `data/siteContent.ts`
- Project case studies: `data/projects.ts`
- Shared UI components: `components/`

## Run locally

```bash
npm install
npm run dev
```
